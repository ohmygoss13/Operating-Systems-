Booting up a Computer 

Booting up a computer is a sequence to functions that start the computer up.  
This is the initial series of events that happen when the computer is turned on.
Booting up a computer can be initiated by the touch of a button or a software command. 

Memory Managements 
Memory management is the process of coordinating computer memory. 
The hardware that is used for memory management called RAM.  
Memory management helps to maintain the overall system.  

Device Controlling 
The operating system controls the devices plugged into the computer.
These items could be a printer, mouse or keyboard. 
These are essential items used daily for a computer.
